# Competitive Programming Templates

A collection of my personal **Competitive Programming (CP) templates** used for coding contests and Data Structures & Algorithms (DSA) practice.

These templates help me quickly reuse commonly used algorithms and data structures during programming contests such as **Codeforces, ICPC, and LeetCode contests**.

---

## 📂 Contents

This repository contains templates for commonly used algorithms and techniques in competitive programming, including:

### Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra’s Algorithm
- Bellman–Ford Algorithm
- Floyd–Warshall Algorithm
- Topological Sort
- Minimum Spanning Tree (Kruskal / Prim)
- Disjoint Set Union (Union-Find)
- Lowest Common Ancestor (Binary Lifting)

### Data Structures
- Segment Tree
- Fenwick Tree (Binary Indexed Tree)
- Sparse Table
- Trie
- Ordered Set (PBDS)

### Number Theory
- Sieve of Eratosthenes
- Fast Exponentiation (Binary Exponentiation)
- Modular Inverse
- Extended Euclidean Algorithm
- nCr with Modulo

### String Algorithms
- KMP Algorithm
- Z Algorithm
- Rolling Hash
- Trie

### Dynamic Programming & Miscellaneous
- Kadane’s Algorithm
- Bitmask DP
- Two Pointers / Sliding Window
- Meet in the Middle
